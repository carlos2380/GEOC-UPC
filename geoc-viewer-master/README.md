# GEOC Viewer

GEOC viewer that actually compiles on a current machine.

### Requirements

* make
* cmake
* gcc
* qt4 (qt5 not supported)
* boost (including dev headers)

### Build

	./build

### Run

To run `ex1`, for example:

	./run bin/ex1

If you wish to run it through Valgrind:

	./run valgrind bin/ex1
